# Kraken X60 CAD Asset

`WCP-0940.STEP` is the canonical source CAD file for the Kraken X60 powered by
Talon FX motor.

Use this asset as the source for the visualizer's `kraken_x60` motor body. Do
not load the STEP file directly at runtime. Runtime meshes derived from this CAD
source live under `assets/meshes/motors/kraken_x60/`.

Current source details:

- File: `WCP-0940.STEP`
- STEP flavor: AP214
- Exporter metadata: SolidWorks 2022 / SwSTEP 2.0
- Source size: about 15 MB
